# TASK-0104 — Модель сессии и флоу состояний

## Описание

Модель сессии гостя и машина состояний цикла: старт → согласие → съёмка → локация → генерация → выбор → печать → выдача → сброс.

## Критерии приёмки

- [ ] Состояния и переходы реализованы
- [ ] Таймауты и автосброс по бездействию
- [ ] Каждый переход пишется в events_log

## Технические заметки

- Автосброс — ключ к работе в потоке гостей
