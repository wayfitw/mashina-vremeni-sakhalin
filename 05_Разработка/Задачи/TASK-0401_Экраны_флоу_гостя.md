# TASK-0401 — Экраны флоу гостя

- **Эпик:** [EPIC-04](../Эпики/EPIC-04_Киоск_UX_и_админ.md) · **Роль:** Frontend · **Оценка:** L · **Приоритет:** Критический · **Статус:** Планируется

## Описание

Полный UX гостя: приветствие «Путешествуй по Сахалину» → выбор локации → съёмка (подсказки) → ожидание → выбор варианта → печать → выдача.

## Критерии приёмки

- [ ] Все экраны реализованы и связаны
- [ ] Индикатор прогресса на ожидании генерации
- [ ] Крупные тач-цели, читаемость на расстоянии

## Технические заметки

- Удержание внимания на 20–40 с генерации
