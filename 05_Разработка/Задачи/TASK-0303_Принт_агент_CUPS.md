# TASK-0303 — Принт агент CUPS

- **Эпик:** [EPIC-03](../Эпики/EPIC-03_Композитинг_и_печать.md) · **Роль:** DevOps/Backend · **Оценка:** L · **Приоритет:** Критический · **Статус:** Планируется

## Описание

Локальный принт-агент: приём готовой карточки → печать borderless на сублимационном принтере, очередь и статусы.

## Критерии приёмки

- [ ] Печать в формате карточки без искажений полей
- [ ] Очередь печати и статусы
- [ ] Ошибки принтера/расходников видны оператору

## Технические заметки

- CUPS/SDK принтера; AirPrint — запасной вариант
