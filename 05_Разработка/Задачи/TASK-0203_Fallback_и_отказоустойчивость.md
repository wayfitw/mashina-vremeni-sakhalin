# TASK-0203 — Fallback и отказоустойчивость

- **Эпик:** [EPIC-02](../Эпики/EPIC-02_Шлюз_генерации_и_пайплайн.md) · **Роль:** Backend · **Оценка:** M · **Приоритет:** Высокий · **Статус:** Планируется

## Описание

Автоматический fallback: основной провайдер недоступен → Replicate → локальный GPU. Обработка ошибок без «зависания».

## Критерии приёмки

- [ ] Обрыв основного провайдера → авто-переключение
- [ ] Гостю показывается корректный статус/повтор
- [ ] Событие fallback логируется
