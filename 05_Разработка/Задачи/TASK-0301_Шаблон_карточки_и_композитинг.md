# TASK-0301 — Шаблон карточки и композитинг

- **Эпик:** [EPIC-03](../Эпики/EPIC-03_Композитинг_и_печать.md) · **Роль:** Backend · **Оценка:** M · **Приоритет:** Критический · **Статус:** Планируется

## Описание

Детерминированный композитинг: выбранный кадр → шаблон карточки + фиксированные логотипы партнёров (до 3) + подпись.

## Критерии приёмки

- [ ] Логотипы всегда пиксель-в-пиксель по шаблону
- [ ] Поддержка до 3 логотипов + подписи
- [ ] Вывод в разрешении под печать

## Технические заметки

- Pillow/sharp/Canvas; логотипы — PNG с прозрачностью
- Логотипы НЕ генерируются моделью (ADR-1)
