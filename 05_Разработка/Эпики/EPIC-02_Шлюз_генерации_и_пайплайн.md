# EPIC-02 — Шлюз генерации и ИИ-пайплайн

- **Этап 2 (Разработка)** · **Приоритет:** Критический · **Статус:** Планируется
- **Связь с ТЗ:** §5.2, §6 ТЗ

## Цель

Реализовать шлюз генерации с провайдерами (Gemini/Replicate/GPU), генерацию 2–3 вариантов, fallback, хранилище локаций и пресетов, приём фото (iPad + QR).

## В объёме

- Шлюз с коннекторами и единым интерфейсом
- Генерация 2–3 вариантов
- Автоматический fallback
- Хранилище локаций/пресетов; приём фото

## Definition of Done

- Генерация «гость + локация» в целевые сроки
- Fallback работает при обрыве связи
- Пресеты локаций управляются без пересборки
