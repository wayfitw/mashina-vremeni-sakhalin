# Фоновое видео приветственного экрана

Положи сюда файл **`intro.mp4`** — он зациклится фоном на первом экране (облёты локаций Сахалина).

## Требования к файлу
- Формат: **MP4 (H.264)**, имя строго `intro.mp4`.
- Ориентация: горизонтальная, под экран стенда (напр. 1920×1080).
- Длительность: 20–60 сек, склейка облётов маяка Анива, бухты Тихой, мыса Великан и т.д.
- **Без звука** (экран стартовый, звук не нужен — видео стартует в muted-режиме).
- Вес: желательно до ~30–50 МБ, чтобы быстро грузилось.

## Как подхватится
Просто положи `intro.mp4` в эту папку и обнови вкладку — код уже ссылается на `media/intro.mp4`.
Пока файла нет — фон остаётся тёмным (флоу работает без видео).
